# mmenanno/unraid-templates

Unraid Community Applications templates for containers I publish.

| Template | Project | GHCR image |
| -------- | ------- | ---------- |
| [`reddit_chat_bridge.xml`](./reddit_chat_bridge.xml) | [reddit_chat_bridge](https://github.com/mmenanno/reddit_chat_bridge) | `ghcr.io/mmenanno/reddit_chat_bridge:latest` |

## Why a separate repo?

Per [Squid's Docker FAQ](https://forums.unraid.net/topic/57181-docker-faq/#comment-566084), the CA submission policy asks template XML to live in a repo separate from the application source so the appfeed scans only template changes, not unrelated commits to the upstream project.
